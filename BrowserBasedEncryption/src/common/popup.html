What is Message Protector (MP)?

MP is a browser extension that lets Alice easily encrypt a message for Bob. MP supports manual encryption so the user sees the encrypted message and can trust that it is encrypted before sharing the message.

Getting Started with MP

Previous |


Steps to encrypt a message and send it through Gmail

  1. Select the encryption tab in MP
  2. Enter the recipient’s email address (only recipient can read the message)
  3. Edit your message in the compose area
  4. Click the “Encrypt” button (message will be encrypted)
  5. Click the “Copy to Clipboard” button
  6. Go to Gmail to compose a message to the recipient
  7. Paste the MP encrypted message into your Gmail message

Steps to decrypt a message using MP

  1. Go to Gmail and copy an encrypted message
  2. Go to MP and select the decryption tab
  3. Paste the encrypted message into MP (use ctrl-v or right click->paste)
  4. Click Enter or click the "Decrypt" button

Security Details

Previous |


Who can read the encrypted message?

The encryption is based on email addresses. If Alice encrypts a message for Bob, only someone that can access Bob’s email account will be able to read the message. The message is as secure as the email account.


What encryption algorithm is used?

Messages are encrypted using 256-bit AES, the U.S. government standard for symmetric encryption.


Who manages the encryption keys?

The MP server generates and distributes encryption keys using an email-based authentication system to limit each user to only the keys associated with their email address. Even though the MP server has the encryption keys, all encryption happens in the browser and the encrypted messages are never sent to the MP server.
